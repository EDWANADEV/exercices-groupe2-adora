# Projets Pratiques Frontend (HTML/CSS/JS)
## Description

Ce dépot regroupe plusieurs minip-projets dévéloppés dans le but de maîtriser les bases du développement web et de la manipulation du Dom.
Chaque dossier contient un exercice indépendant:
- Une calculatrice interactive
    - Conçus simplement elle résoud les oppérations de bases(addition, soustraction, multiplication, division).
    - Présente à son interface un bouton C pour la surpression et une touche = pour afficher le résultat à l'écran.
- Un formulaire de validation
    - Il présente trois champs qui doivent être rempli par l'utilisateur.
    - La validation du formulaire est gérée par le html et le JavaScript .
    - IL affiche  un message de validation lorsqu'il est validé.
- Un carrousel d'images
    - IL présente une image toute les trois secondes géré par le JavaScript
    - Présente des boutons pour permettre à l'utilisateur de faire défiler les images à son aise.
- Une liste de tâches
    - Elle présente un champ avec un bouton ajouter pour ajouter la tâche
    - Une fois la tâche ajouter, elle peut être marquée comme terminée ou surpprimer.
    - l'utilisateur ne peut pas ajouter plus de huit tâches.

Pour ces minis-projet, le html nous à permit de structurer les pages, le CSS de les stylisées et le JavaScript de les rendre dynamique.
